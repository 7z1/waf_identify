# WAF 识别

整合4款waf识别工具

1. sqlmap
2. waf00f
3. wafid
4. DirBrute



共300+waf指纹规则

方便自行添加指纹

方便移植接入自己工具

主要提取自sqlmap

使用命令：python main.py http://www.xxx.com
